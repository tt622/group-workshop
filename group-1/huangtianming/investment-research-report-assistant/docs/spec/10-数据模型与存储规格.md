## 1. 概念模型

```
Session (会话)
   │ 1
   ├── * QARecord      (问答记录)
   ├── * Report        (研报) ── 1 ── ReportSummary (结构化摘要)
   │                           │
   │                           └── * Citation (原文引用片段)
   └── * CompareResult (对比结果) ── * ComparedField

Stock (股票) — 跨会话共享，仅按 code 索引
```

**核心关系**：
- `QARecord.session_id` → `Session.session_id`（FK 逻辑引用）
- `Report.session_id` → `Session.session_id`
- `QARecord.citations[].report_id` → `Report.report_id`
- `CompareResult.session_id` → `Session.session_id`；`CompareResult.report_ids[]` → `Report.report_id`

## 2. 存储引擎

| 项 | 说明 |
|----|------|
| 引擎 | JSON 文件（`json` 标准库）+ 本地文件系统 |
| 编码 | UTF-8，缩进 2 空格 |
| 读写模式 | RMW（全量读入 → 修改 → 全量写回） |
| 并发安全 | 无锁，不支持并发写入（教学环境） |
| 文件布局 | 见 `08` §8.2 |

## 3. 实体定义 — Session

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `session_id` | string (UUID) | 是 | **PK** | 会话唯一标识 |
| `title` | string | 是 | ≤ 23 字符 | 默认 `"新会话"`，首次问答自动改名 |
| `created_at` | string (ISO-8601) | 是 | UTC+Z | 创建时间 |
| `updated_at` | string (ISO-8601) | 是 | UTC+Z | 最后更新时间 |
| `query_count` | integer | 是 | ≥ 0 | 累计问答次数 |

## 4. 实体定义 — QARecord

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `id` | string | 是 | **PK**，`rec_{ts}` | 基于时间戳生成 |
| `session_id` | string | 是 | **FK** → Session | 所属会话 |
| `query` | string | 是 | 1–500 字符 | 用户提问 |
| `answer` | string | 是 | — | 回答正文 |
| `llm_used` | boolean | 是 | 规则-005 | 是否使用真实 LLM |
| `model` | string\|null | 是 | — | 模型标识 |
| `response_time_ms` | integer | 是 | SLO < 5000 | 响应耗时 |
| `answer_source` | string | 是 | `copaw\|bailian\|demo` | 来源标识 |
| `citations` | Citation[] | 是 | 可为空数组 | 原文引用 |
| `timestamp` | string (ISO-8601) | 是 | UTC+Z | 记录时间 |

## 5. 实体定义 — Report

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `report_id` | string | 是 | **PK**，`rpt_{ts}` | — |
| `session_id` | string | 是 | **FK** → Session | — |
| `filename` | string | 是 | ≤ 128 字符 | 原始文件名 |
| `format` | string | 是 | `doc\|docx\|ppt\|pptx\|pdf\|html\|htm` | 文件格式 |
| `size_bytes` | integer | 是 | ≤ 30 MB | 文件大小 |
| `file_path` | string | 是 | 本地路径 | `./data/files/{report_id}.{ext}` |
| `status` | string | 是 | `uploading\|parsing\|ready\|failed` | 解析状态 |
| `failure_reason` | string\|null | 否 | 脱敏 | 解析失败原因 |
| `ocr_used` | boolean | 是 | — | 是否启用 OCR 兜底 |
| `text_content` | string | 否 | 解析成功后写入 | 解析后的文本 |
| `stocks` | string[] | 是 | 可空 | 识别到的股票代码 |
| `uploaded_at` | string | 是 | ISO-8601 | — |
| `parsed_at` | string\|null | 否 | ISO-8601 | — |

## 6. 实体定义 — ReportSummary

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `report_id` | string | 是 | **PK/FK** → Report | — |
| `rating` | string\|null | 是 | 无法识别则 null | 评级 |
| `target_price` | string\|null | 是 | 保留原币种表述 | 目标价 |
| `core_views` | string[] | 是 | 可空数组 | 核心观点要点 |
| `citations` | object | 是 | 每个字段的原文引用 | 见 `09` §11 |
| `extracted_at` | string | 是 | ISO-8601 | — |

## 7. 实体定义 — Citation

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `report_id` | string | 是 | FK → Report | — |
| `snippet` | string | 是 | ≤ 300 字符 | 原文片段 |
| `location` | string\|null | 否 | 如 `p.3` | 页码/段落 |

## 8. 实体定义 — CompareResult

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `compare_id` | string | 是 | **PK**，`cmp_{ts}` | — |
| `session_id` | string | 是 | FK → Session | — |
| `report_ids` | string[] | 是 | length ≥ 2 | 参与对比的研报 |
| `fields` | ComparedField[] | 是 | — | 字段聚合结果 |
| `elapsed_ms` | integer | 是 | SLO < 120 000 | 生成耗时 |
| `created_at` | string | 是 | ISO-8601 | — |

**ComparedField**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `field` | string | `rating / target_price / core_views` |
| `values` | `{report_id, value}[]` | 每份研报的取值 |

## 9. 实体定义 — StockSnapshot（DTO，不落盘）

| 字段 | 类型 | 说明 |
|------|------|------|
| `code` | string | 带交易所后缀 |
| `name` | string | 名称 |
| `exchange` | string | `SH \| SZ \| BJ \| HK \| US` |
| `latest_price` | number | 最新价 |
| `change_pct` | number | 涨跌幅（%） |
| `pe_ttm` | number\|null | TTM PE |
| `market_cap` | string | 市值文案 |
| `updated_at` | string | ISO-8601 |

## 11. 关键业务逻辑

| 逻辑 | 触发条件 | 行为 |
|------|----------|------|
| 首次问答自动命名 | `query_count` 从 0 → 1 | `title = query[:20] + ("..." if len(query)>20 else "")` |
| 会话级联删除 | `delete_session` | 删除全部 QARecord / Report（含文件）/ Summary / CompareResult |
| 研报解析失败隔离 | 单份失败 | 仅该条 `status=failed`，其余继续（AC-002-03） |
| 提取字段兜底 | LLM 未返回某字段 | `null` 或空数组（AC-003-01） |
| 对比缺失字段 | 某研报缺失该字段 | `value=null`，MUST NOT 代填（AC-005-03） |
| 引用为空 | 问答未命中证据 | `citations=[]`，前端展示"未在当前研报中找到依据"（AC-004-03） |

## 12. API 响应 DTO 概要

| API 端点 | 核心 DTO 字段 | 额外注入 |
|----------|---------------|----------|
| `POST /ask` | answer, llm_used, model, response_time_ms, answer_source, citations | `traceId` |
| `POST /sessions` | session_id, title, created_at, query_count | `traceId` |
| `GET /sessions` | sessions[] | `traceId` |
| `GET /sessions/<id>/records` | records[] | `traceId` |
| `POST /sessions/<id>/reports` | report_id, status, format, size_bytes | `traceId` |
| `GET /reports/<id>/summary` | rating, target_price, core_views, citations | `traceId` |
| `POST /sessions/<id>/compare` | compare_id, fields, elapsed_ms | `traceId` |
| `GET /stocks` | code, name, latest_price, change_pct, … | `traceId` |
