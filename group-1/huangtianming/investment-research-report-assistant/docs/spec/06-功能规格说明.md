## 1. 页面总体布局

```
┌────────────────────────────────────────────────────────────────┐
│  Header: 标题"投研问答助手" + [能力状态芯片] + [数据不出域提示]   │
├────────────┬───────────────────────────────────┬───────────────┤
│  Sidebar   │   Main 工作区                     │  Right Panel  │
│  会话列表   │   ┌───────────────────────────┐   │  股票详情面板  │
│  [+新建]   │   │ 研报列表 + 上传区（顶部）   │   │  ────────    │
│  session-1 │   │   · 上传按钮  · 解析状态    │   │  · 代码/名称  │
│  session-2 │   │   · [选择对比]             │   │  · 基础信息   │
│    ...     │   ├───────────────────────────┤   │  · 行情摘要   │
│            │   │ 结构化摘要卡片区            │   │              │
```

## 2. Header 区域

### 2.1 能力状态芯片（Capability Chips）

| 条件 | 芯片显示 | 样式 | 说明 |
|------|----------|------|------|
| `caps.copaw_configured = true` | CoPaw 桥接 | 绿色 | CoPaw 已配置 |
| `caps.bailian_configured = true` | 百炼 · {model} | 蓝色 | 百炼已配置 + 模型名 |
| 均未配置 | 离线演示 | 灰色 | 降级到 Demo 模式 |
| 解析服务异常 | 解析异常 | 红色 | 上传暂不可用 |

### 2.2 固定提示

- "数据不出域" 常驻标识（US-007 相关）。

## 3. Sidebar 会话管理（对齐 US-001）

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 加载列表 | 页面首次渲染 | `GET /sessions` | 渲染会话列表 + 默认选中第一个 |
| 新建会话 | 点击"+ 新建" | `POST /sessions` | 插入列表头部 + 自动选中空会话 |
| 切换会话 | 点击会话项 | `GET /sessions/<id>/records`、`GET /sessions/<id>/reports` | 仅展示该会话的研报、问答、对比结果 |
| 删除会话 | 点击 × | `DELETE /sessions/<id>` | 二次确认弹窗 → 从列表移除 + 清空主区 |

**对齐 AC**：AC-001-01/02/03/04。

## 4. 研报上传与解析区（对齐 US-002）

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 上传研报 | 拖拽 / 点击上传 | `POST /sessions/<id>/reports` | 支持 `.doc/.docx/.ppt/.pptx/.pdf/.html` |
| 解析状态 | 轮询或 SSE | `GET /reports/<id>` | 状态：`uploading / parsing / ready / failed` |
| 失败重试 | 点击"重试" | `POST /reports/<id>/reparse` | 失败原因气泡提示，不影响其他研报 |
| OCR 兜底 | PDF 文本层缺失自动触发 | 后端内部 | 前端展示"已启用 OCR" 标识 |

**文件格式与大小约束**：见 `07` §5。

## 5. 结构化摘要卡片区（对齐 US-003）

| 元素 | 内容 | 交互 |
|------|------|------|
| 评级（rating） | 文本徽章 | 点击高亮对应原文片段 |
| 目标价（target_price） | 数值 + 币种 | 点击高亮对应原文片段 |
| 核心观点（core_views） | 2–5 条要点列表 | 每条可展开原文引用 |
| 未识别字段 | 灰色"未提取到"占位 | 不得隐藏字段（AC-003-01） |

## 6. 问答对话区（对齐 US-004） — 三态渲染

| 状态 | 条件 | 显示内容 |
|------|------|----------|
| **A 空状态** | `currentSession === null` | "请创建或选择一个会话开始" |
| **B 常见问题** | 有会话 && 无 QA 记录 | 4 张快速卡片：如"此研报评级如何？""核心投资逻辑？""目标价依据？""潜在风险？"，点击自动填入输入框 |
| **C 对话历史** | 有 QA 记录 | 按时间顺序渲染用户问题 + 回答卡片；每张卡片含 `answer`、来源标签、`citations[]` 原文片段、`response_time_ms` |

### 6.1 回答卡片来源标签

| `answer_source` | `llm_used` | 显示文本 | 样式 |
|-----------------|------------|----------|------|
| `copaw` | true | CoPaw | 绿色 |
| `bailian` | true | 百炼 · {model} | 蓝色 |
| `demo` | false | 离线演示 | 灰色 |

### 6.2 无证据时的文案

当 `citations` 为空，必须固定显示："未在当前研报中找到依据"（不得伪造引用，AC-004-03）。

## 7. 多研报对比抽屉（对齐 US-005）

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 选中研报 | 研报列表复选框 | — | ≥ 2 份方可激活"对比"按钮 |
| 生成对比 | 点击"对比" | `POST /sessions/<id>/compare` | 底部抽屉展示对比表 |
| 对比表列 | — | — | `字段 \| 研报 A \| 研报 B \| ...`，每列顶部展示研报名 |
| 缺失字段 | — | — | 展示"未提取到"，不得代填（AC-005-03） |
| 耗时提示 | — | — | 展示 `elapsed_ms`，目标 < 120 000 ms（AC-005-04） |

## 8. 股票详情面板（对齐 US-006）

| 功能 | 触发 | API 调用 | 行为 |
|------|------|----------|------|
| 从研报调起 | 点击研报中识别到的股票标签 | `GET /stocks?code=...` | 右侧面板展开展示信息 |
| 主动搜索 | 输入代码 / 名称 | `GET /stocks/search?q=...` | 下拉候选列表 |
| 多标的消歧 | 候选 > 1 | — | 用户必须显式选择（AC-006-03） |
| 基础信息 | — | — | 代码、名称、所属行业、最新价、涨跌幅等 |

## 9. 输入区域

| 元素 | 类型 | 行为 |
|------|------|------|
| query-input | `<textarea>` | 3 行，placeholder "请针对本会话已导入的研报提问…" |
| char-counter | `<span>` | 实时显示 `x / 500`，超长置红 |
| send-btn | `<button>` | loading 时 disabled + 文案"发送中…" |
| clear-btn | `<button>` | 清空输入框 |

**输入约束**：1 ≤ length ≤ 500；无会话或无已导入研报时发送按钮禁用（AC-004-01）。

## 10. 错误处理

| 后端 `error.code` | HTTP | 前端展示 |
|--------------------|------|----------|
| `EMPTY_QUERY` | 400 | 行内提示"请输入问题" |
| `INVALID_QUERY` | 400 | 行内提示"问题长度必须在 1–500 字符之间" |
| `SESSION_NOT_FOUND` | 404 | Toast "会话不存在，请重新选择" |
| `REPORT_PARSE_FAILED` | 422 | 研报卡片红色状态 + "查看原因" |
| `UNSUPPORTED_FORMAT` | 415 | Toast "仅支持 WORD/PPT/PDF/HTML" |
| `STOCK_NOT_FOUND` | 404 | 面板展示"未找到匹配股票" |
| `UPSTREAM_ERROR` | 500 | Toast "外部服务暂不可用，请稍后重试"（US-007） |

## 11. 状态管理（React useState）

| State 变量 | 类型 | 初始值 | 用途 |
|------------|------|--------|------|
| `sessions` | `Session[]` | `[]` | 会话列表 |
| `currentSession` | `Session \| null` | `null` | 当前选中会话 |
| `reports` | `Report[]` | `[]` | 当前会话研报列表 |
| `selectedReportIds` | `string[]` | `[]` | 对比选中项 |
| `summary` | `Summary \| null` | `null` | 当前研报结构化摘要 |
| `records` | `QARecord[]` | `[]` | 当前会话问答历史 |
| `compareResult` | `CompareResult \| null` | `null` | 对比结果 |
| `stockDetail` | `StockSnapshot \| null` | `null` | 右侧面板展示的股票 |
| `loading` | `{ask,parse,compare,stock}` | `{}` | 各操作 loading 态 |
| `caps` | `Capabilities` | `{}` | 能力探测结果 |
