## 5. 依赖链与风险

**依赖链**：

```
S1 会话骨架
   │
   ▼
S2 研报解析 + 结构化（依赖会话上下文）
   │
   ▼
S3 问答 / 对比 / 股票（依赖研报 ready 状态）
   │
   ▼
S4 异常兜底 + 收口（依赖 S1~S3 主链路稳定）
```

**风险表**：

| 风险 | 影响 | 应对 |
|------|------|------|
| 外部 LLM 不可用 | 问答降级 | Demo 模式兜底 + 降级顺序固化 |
| PDF 格式多样，解析失败率高 | 提取准确率不达标 | OCR 兜底 + 单份失败隔离 |
| 多研报对比超过 2 分钟 | AC-005-04 不达标 | 并行读取 + 字段级缓存 + 性能专项 |
| 股票外部接口不稳 | US-006 可用性 | 超时重试 + 缓存最近结果 + 明确"接口暂不可用"提示 |
| LLM 回答幻觉 | 伪造引用，违反 AC-004-03 | 强制 citations 非空校验 + 无证据固定文案 |
