## 1. 性能需求

### 1.1 延迟 SLO

| 指标 | 目标值 | 度量方式 | 来源 |
|------|--------|----------|------|
| 问答端点 P95 延迟 | < **5 000 ms** | 响应体 `response_time_ms` | `05` AC-004 |
| 研报解析 P95 耗时（单份 < 10MB） | < **30 000 ms** | 后端解析日志 | `03` §4 |
| 研报 OCR 兜底 P95 耗时 | < **90 000 ms** | 后端 OCR 日志 | `03` §5 风险 |
| 多研报对比生成 P95 耗时 | < **120 000 ms** | `compareResult.elapsed_ms` | `05` AC-005-04 |
| 会话管理端点 P95 延迟 | < **500 ms** | 服务端计时 | JSON I/O < 10ms |
| 股票查询端点 P95 延迟 | < **1 500 ms** | 服务端计时 | `05` AC-006 |

### 1.2 降级策略

```
CoPawAgent.ask(query, session_id)
  ├─[1] CoPaw 桥接    → answer_source='copaw',   llm_used=true
  ├─[2] 百炼 DashScope → answer_source='bailian', llm_used=true
  └─[3] 离线演示兜底   → answer_source='demo',    llm_used=false
```

| 约束 | 说明 |
|------|------|
| 三级降级**不可跳级** | 必须按 CoPaw → 百炼 → Demo 顺序尝试 |
| 降级**静默执行** | 不抛异常，通过 `answer_source` 标识 |
| Demo 模式**始终可用** | 无外部依赖，纯模板拼接 |
| 降级结果 **必须** 保留 `citations` 占位 | 为空时前端展示"未在当前研报中找到依据" |

### 1.3 吞吐量

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 单实例并发会话数 | ≥ 10 | 教学环境 |
| 单会话研报上限 | ≤ 20 份 | 超出提示"建议新建会话" |

## 2. 可用性需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 服务可用性 | 99%（教学环境） | 单实例 Flask |
| MTTR | < 5 分钟 | 重启 Flask 进程即可恢复 |
| 研报解析失败隔离 | 100% | 单份失败 MUST NOT 影响其他研报（AC-002-03） |
| 外部 LLM 故障可用性 | US-007 | 必须返回可理解提示，会话与研报数据保留 |

## 3. 可观测性需求

### 3.1 链路追踪

| 项 | 规格 |
|----|------|
| traceId 格式 | `tr_{uuid.hex}`（32 位十六进制） |
| 注入位置 | 所有成功响应顶层 + 错误响应 `error.traceId` |
| 上游透传 | 优先复用请求头 `X-Trace-Id`，缺省时本地生成 |

### 3.2 日志字段

| 字段 | 说明 |
|------|------|
| `traceId` | 同响应体 |
| `session_id` | 会话上下文 |
| `endpoint` | API 路径 |
| `elapsed_ms` | 后端处理耗时 |
| `answer_source` | 问答来源（仅 /ask） |
| `report_id` | 研报上下文（仅解析相关端点） |

### 3.3 错误码体系（与 `09` 一致）

| HTTP | `error.code` | 触发条件 |
|------|--------------|----------|
| 400 | `EMPTY_QUERY` | query 为空/空白/null |
| 400 | `INVALID_QUERY` | query > 500 字符 或 缺少 session_id |
| 404 | `SESSION_NOT_FOUND` | session_id 不存在 |
| 404 | `REPORT_NOT_FOUND` | report_id 不存在 |
| 404 | `STOCK_NOT_FOUND` | 股票代码/名称查不到 |
| 415 | `UNSUPPORTED_FORMAT` | 非 WORD/PPT/PDF/HTML |
| 422 | `REPORT_PARSE_FAILED` | 解析失败 |
| 500 | `UPSTREAM_ERROR` | Agent 编排内部异常 |

## 4. 安全需求

| 项 | 当前实现 | 说明 |
|----|----------|------|
| API 认证 | 无（开放访问） | Workshop 教学版 |
| CORS | 全开放 | 生产需白名单 |
| 速率限制 | 无 | 教学场景不限速 |
| 数据存储 | 本地 JSON + 本地文件系统 | **不出域**（R-09） |

### 4.1 敏感数据保护

| 数据类型 | 保护方式 |
|----------|----------|
| LLM API Key | 环境变量 `.env`，禁止硬编码 |
| 用户查询 / 问答记录 | 仅本地 JSON，除 LLM 调用外不传输至第三方 |
| 研报原文 | 仅存储在本地 `./data/reports/` 目录，禁止外传 |
| 外部模型错误信息 | MUST 脱敏后返回（AC-007-03），不得暴露调用栈 / URL / Key |

## 5. 数据约束

### 5.1 输入约束

| 约束 | 规格 | 来源 |
|------|------|------|
| `query` 长度 | 1 – 500 字符 | `04` R-02 |
| `session_id` 必填 | 非空 UUID 字符串 | `04` R-01 |
| 会话标题默认值 | `"新会话"`（首次问答后自动改名） | `10` §6 |
| 单研报文件大小 | ≤ 30 MB | 上传约束 |
| 单会话研报数 | ≤ 20 份 | §1.3 |
| 研报文件格式 | `.doc / .docx / .ppt / .pptx / .pdf / .html / .htm` | `04` R-04 |

### 5.2 类型约束

| 字段 | 类型 | 枚举 / 约束 |
|------|------|-------------|
| `llm_used` | boolean | 必须为 true/false，不得为字符串 |
| `answer_source` | string | `copaw \| bailian \| demo` |
| `report.status` | string | `uploading \| parsing \| ready \| failed` |
| `rating` | string\|null | 无法识别 → null |
| `target_price` | string\|null | 保留原币种表述 |
| `core_views` | string[] | 无法识别 → 空数组 |
| `citations` | object[] | 元素含 `report_id` + `snippet` + `location?` |

## 6. 可维护性
