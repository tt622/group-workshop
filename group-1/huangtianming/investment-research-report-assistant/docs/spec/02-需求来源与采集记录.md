### 痛点记录

### 需求记录

1. **研报解析**：支持 WORD/PPT/PDF/HTML 四种格式，自动提取标题、评级、目标价、核心观点
2. **研报问答**：基于研报内容进行自然语言问答，支持引用原文
3. **多研报对比**：一键生成同主题/同标的多份研报的关键指标对比
4. **股票信息查询**：从研报上下文直达相关股票基础与行情信息
5. **降级保障**：外部 LLM 不可用时保持核心体验不中断（远期）

## S-02 邮件沟通 — 关于投研助手的需求建议

| 项 | 内容 |
|---|---|
| 发件人 | 李总监 <li.d@company.com> |
| 收件人 | 王经理 <wang.m@company.com> |
| 时间 | 2026-04-10 09:30 |

**问题 1 — 研报阅读效率低**：每天需阅读 10+ 份研报，平均每份 30-50 页，耗时 2-3 小时。希望能快速提取关键信息（评级、目标价、核心观点）。

**问题 2 — 研报对比困难**：对比不同券商对同一标的的研报需手动复制粘贴到 Excel，效率低下，希望能自动生成对比表。

**问题 3 — 股票数据分散**：查股票数据要在多个系统之间切换，希望能在研报阅读界面直接查到相关股票数据，覆盖"读报 → 对比 → 查数据"的连续动作。

## S-03 用户访谈 — 高级分析师

| 项 | 内容 |
|---|---|
| 访谈对象 | 陈研究员（高级分析师） |
| 访谈人 | 张产品 |
| 时间 | 2026-04-12 10:00-11:00 |
| 方式 | 线上访谈 |

| 问题 | 回答要点 |
|------|----------|
| Q1：平时工作中最耗时的环节？ | 读研报最耗时。每天扫一遍行业研报找重点，电子版反而不知从哪看起。 |
| Q2：最希望在研报中获取什么信息？ | ① 评级和目标价 → ② 核心观点（分析师主要逻辑）→ ③ 数据预测（营收增速、PE 等） |
| Q3：同时看多份研报如何对比？ | 各家格式不同，需手动整理到 Excel。希望能自动生成对比表。 |
| Q4：研报格式是否统一？ | 不统一：WORD/PPT/PDF/HTML 都有，有些 PDF 是扫描版，文本层缺失。 |
| Q5：对 AI 的了解或期待？ | ChatGPT 用得多，但涉及内部数据不敢传。**必须内部部署，数据安全第一。** |

**总结**：① 快速提取 + 自动对比为核心诉求 ② 必须本地部署，数据不出域 ③ 支持多格式解析（含 OCR 兜底）④ AI 接受度高，但需对回答可追溯。
