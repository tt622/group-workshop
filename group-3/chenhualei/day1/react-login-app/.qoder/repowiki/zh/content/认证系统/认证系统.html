简介

React Login App 的认证系统实现了完整的用户登录、登出和会话管理功能。系统采用 Zustand 进行状态管理，结合 localStorage 实现会话持久化。

系统架构

graph TB subgraph "认证系统" A[LoginForm] --> B[authStore] B --> C[localStorage] B --> D[ProtectedRoute] D --> E[DashboardPage] D --> F[TetrisPage] end

核心组件

  • LoginForm: 登录表单组件
  • authStore: 认证状态管理
  • ProtectedRoute: 路由保护组件

功能特性

用户登录

  • 用户名密码验证
  • 表单输入验证
  • 错误提示

会话管理

  • 登录状态持久化
  • 自动会话恢复
  • 安全登出

路由保护

  • 未认证用户重定向
  • 受保护页面访问控制

安全机制

  • 输入验证: 使用 Zod 进行表单验证
  • 状态隔离: 认证状态集中管理
  • 会话持久化: 安全的本地存储
