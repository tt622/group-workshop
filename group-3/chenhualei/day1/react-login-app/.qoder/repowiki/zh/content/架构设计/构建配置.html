简介

React Login App 使用 Vite 作为构建工具，提供快速的开发体验和高效的生产构建。

Vite 配置

基础配置

// vite.config.js
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
});
代码来源
  • vite.config.js

配置说明

  • @vitejs/plugin-react: 提供 React 支持、JSX 转换和热重载

构建脚本

// package.json
{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview",
    "lint": "eslint ."
  }
}

脚本说明

  • dev: 启动开发服务器，支持热重载
  • build: 构建生产版本，输出到 dist 目录
  • preview: 预览生产构建结果
  • lint: 运行 ESLint 代码检查

开发服务器

flowchart LR A[源代码] --> B[Vite Dev Server] B --> C[ESM 转换] C --> D[浏览器] E[文件变更] --> F[HMR] F --> B

特性

  • 即时启动: 无需打包，直接启动
  • 热重载: 代码变更即时更新
  • ESM 支持: 原生 ES 模块加载

生产构建

flowchart TD A[源代码] --> B[代码转换] B --> C[代码压缩] C --> D[资源优化] D --> E[输出到 dist]

构建优化

  • 代码压缩: 自动压缩 JavaScript 和 CSS
  • Tree Shaking: 移除未使用的代码
  • 资源优化: 图片和静态资源优化
  • 代码分割: 自动代码分割优化加载
