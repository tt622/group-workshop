环境要求

  • Node.js: 18.0.0 或更高版本
  • npm: 8.0.0 或更高版本（或 yarn/pnpm）
  • 浏览器: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

安装步骤

1克隆项目
git clone <repository-url>
cd react-login-app
2安装依赖
npm install

这将安装所有必要的依赖包，包括 React、Vite、React Router、Zustand 等。

3配置环境（可选）

项目使用默认配置即可运行，无需额外环境变量配置。

启动开发服务器

4启动开发环境
npm run dev

开发服务器将在 http://localhost:5173 启动，支持热重载功能。

登录测试

项目包含模拟的登录功能，可以使用以下测试凭据：

  • 用户名: admin
  • 密码: password

或者使用：

  • 用户名: user
  • 密码: 123456

游戏功能

登录后可以访问俄罗斯方块游戏，支持以下操作：

  • 方向键: 移动方块
  • 上方向键: 旋转方块
  • 空格键: 快速下落
  • P 键: 暂停游戏

常见问题

端口被占用

如果 5173 端口被占用，Vite 会自动尝试其他端口。也可以手动指定端口：

npm run dev -- --port 3000

依赖安装失败

尝试清理缓存后重新安装：

npm cache clean --force
rm -rf node_modules package-lock.json
npm install

热重载不工作

检查文件保存后是否有重新编译，确保浏览器控制台没有错误信息。
