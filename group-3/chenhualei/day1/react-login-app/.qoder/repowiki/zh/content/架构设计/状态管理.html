简介

React Login App 使用 Zustand 作为状态管理解决方案。Zustand 是一个轻量级、简洁的状态管理库，提供了直观的 API 和优秀的开发体验。

Zustand 概述

为什么选择 Zustand

  • 轻量级: 打包后仅 1KB 左右
  • 简单易用: API 简洁直观，学习成本低
  • 无需 Provider: 不需要包裹 Provider 组件
  • 性能优秀: 支持选择器避免不必要的渲染
  • TypeScript 支持: 完善的类型定义

Store 设计

认证 Store

// authStore.js
import { create } from 'zustand';

const useAuthStore = create((set, get) => ({
  user: null,
  isAuthenticated: false,
  loading: false,
  error: null,

  login: async (username, password) => {
    set({ loading: true, error: null });
    try {
      // 模拟登录验证
      if (username && password) {
        const user = { username };
        localStorage.setItem('user', JSON.stringify(user));
        set({ user, isAuthenticated: true, loading: false });
        return true;
      }
      throw new Error('登录失败');
    } catch (error) {
      set({ error: error.message, loading: false });
      return false;
    }
  },

  logout: () => {
    localStorage.removeItem('user');
    set({ user: null, isAuthenticated: false });
  },

  initialize: () => {
    const user = localStorage.getItem('user');
    if (user) {
      set({ user: JSON.parse(user), isAuthenticated: true });
    }
  },
}));

export default useAuthStore;

使用方式

在组件中使用 Store

import useAuthStore from '../store/authStore';

function LoginPage() {
  const { login, isAuthenticated, loading, error } = useAuthStore();

  const handleSubmit = async (data) => {
    const success = await login(data.username, data.password);
    if (success) {
      navigate('/dashboard');
    }
  };

  // ...
}

选择器优化

使用选择器避免不必要的渲染：

// 只订阅 user 状态变化
const user = useAuthStore((state) => state.user);

// 只订阅特定状态
const login = useAuthStore((state) => state.login);

状态持久化

项目通过 localStorage 实现状态持久化：

  • 登录时: 将用户信息存入 localStorage
  • 页面刷新: 通过 initialize 方法恢复状态
  • 退出登录: 清除 localStorage 数据
