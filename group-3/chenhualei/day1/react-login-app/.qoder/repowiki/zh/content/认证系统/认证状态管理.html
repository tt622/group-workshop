状态结构

{
  user: null | Object,        // 当前用户信息
  isAuthenticated: boolean,   // 认证状态
  loading: boolean,          // 加载状态
  error: null | string       // 错误信息
}

状态操作

login

执行用户登录验证

const login = async (username, password) => {
  set({ loading: true, error: null });
  // 验证逻辑
  // 成功: set({ user, isAuthenticated: true, loading: false })
  // 失败: set({ error, loading: false })
}

logout

执行用户登出

const logout = () => {
  localStorage.removeItem('user');
  set({ user: null, isAuthenticated: false });
}

initialize

从 localStorage 恢复会话

const initialize = () => {
  const user = localStorage.getItem('user');
  if (user) {
    set({ user: JSON.parse(user), isAuthenticated: true });
  }
}

持久化机制

sequenceDiagram participant User as 用户 participant Store as authStore participant Storage as localStorage User->>Store: 登录 Store->>Store: 验证成功 Store->>Storage: 存储用户信息 Store-->>User: 更新状态 User->>Store: 刷新页面 Store->>Storage: 读取用户信息 Storage-->>Store: 返回数据 Store-->>User: 恢复登录状态
