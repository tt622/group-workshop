简介

本项目是一个基于 React 19 和 Vite 的现代化登录认证应用，展示了当前前端开发的最佳实践。项目实现了完整的用户认证流程，包括登录表单验证、路由保护、状态管理和会话持久化等功能。

作为一个功能完整的示例项目，它不仅提供了基础的登录功能，还集成了俄罗斯方块游戏作为额外的娱乐功能，展示了复杂状态管理和游戏逻辑在前端应用中的实现方式。

核心功能

用户认证系统

  • 基于 React Hook Form 和 Zod 的表单验证
  • 使用 Zustand 进行全局状态管理
  • localStorage 会话持久化
  • 受保护路由机制

俄罗斯方块游戏

  • 完整的游戏逻辑实现
  • 七种经典方块类型
  • 幽灵方块预览功能
  • 分数和等级系统

技术栈

核心框架

  • React 19.2.4 - 用户界面库
  • Vite 8.0.4 - 构建工具
  • React Router 7.14.0 - 路由管理

状态管理

  • Zustand 5.0.12 - 轻量级状态管理

表单处理

  • React Hook Form 7.72.1 - 表单状态管理
  • Zod 4.3.6 - 类型安全验证

代码质量

  • ESLint 9.39.4 - 代码检查

架构概览

sequenceDiagram participant User as 用户 participant Login as LoginForm participant Auth as authStore participant Route as React Router participant Page as DashboardPage User->>Login: 输入凭据 Login->>Auth: 调用 login() Auth-->>Login: 返回认证结果 Login->>Route: 导航到仪表板 Route->>Page: 渲染受保护页面 Page-->>User: 显示仪表板

快速开始

安装依赖

npm install

启动开发服务器

npm run dev

构建生产版本

npm run build
